## VSCode

## Flow

- debug(VSCode): 
  - `F5`

- build(VSCode): 
  - `F1`/`Ctrl + Shift + p`
  - `CMake: Clean Rebuild`

- run:
  - `$ ./build/<program:cpp_development_boilerplate>`
